Welcome to my data analytics portfolio.

##  Project 1: Supermarket Sales Dashboard (Power BI) 

**Goal:** Analyse supermarket sales performance and help management take data-driven decisions.

**Dataset:** 3 years of supermarket transactional data  
(Branch, City, Product line, Customer type, Invoice date, Rating, Payment type, etc.)

### Key Analysis
- Sales performance by **Region, City, Branch, Product Category**
- Trends by **Day, Month, Year** and **Time of day**
- Performance by **Customer type & Payment method**
- Top & bottom performing product lines

### KPIs
- **Total Sales**
- **Total Profit**
- **Quantity Sold**
- **Average Rating**
- **Profit Margin %**

### What I Did
- Cleaned and transformed raw CSV data using **Power Query**
- Built a **star-schema data model** with proper relationships
- Created calculated columns and **DAX measures** for KPIs & time intelligence
- Designed an **interactive dashboard** with slicers, drill-through and tooltips
- Added summary insights for quick decision-making

**Tools & Skills:** Power BI, Power Query, DAX, Data Modelling, Data Cleaning, Data Visualization

## Project 2: Sales Forecasting – Machine Learning (Python)

**Goal:** Build a model to forecast future sales and help the business plan inventory and targets.

**Dataset:** Historical sales data with features like Date, Store/Region, Product, Quantity, Revenue, etc.

**Tech Stack:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

**Steps Performed:**
- Performed data cleaning (handling missing values, outliers, date-time formatting)
- Performed EDA to understand seasonality, trends and patterns
- Created new features like Month, Year, Day of week, Holiday flags etc.
- Split data into train–test and built forecasting models (e.g., Linear Regression / RandomForestRegressor)
- Evaluated model using MAE / RMSE and compared different models
- Visualized actual vs predicted sales

**Key Insights / Outcome:**
- Identified peak sales months and low-performing periods
- Helped in understanding which product/store combinations drive maximum revenue
- Built a forecasting solution that can be extended for future data

## Project 3: Amazon Mobile Market Analysis (Excel)

**Goal:** Analyse mobile phone listings on Amazon to understand pricing patterns and customer preferences.

**Dataset:** Mobile phone dataset with Brand, Model, Price, Ratings, Reviews, RAM, Storage, etc.

**Tools:** Excel (Pivot Tables, Charts, Filters, Conditional Formatting)

**Steps Performed:**
- Cleaned and transformed raw data using Excel functions
- Created Pivot Tables for:
  - Brand-wise average price and rating
  - RAM/Storage vs Price comparison
  - Rating distribution across brands
- Built interactive charts and a dashboard for management

**Key Insights / Outcome:**
- Identified top brands with high ratings but competitive pricing
- Found the ideal price range where most mobiles are sold
- Helped in understanding which specs (RAM/Storage) influence price the most
