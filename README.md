## ðŸ›’ Project 1: Supermarket Sales Dashboard (Power BI)

**Goal:** Analyse supermarket sales performance and help management take data-driven decisions.

**Dataset:** 3 years of supermarket transactional data  
(Branch, City, Product line, Customer type, Invoice date, Rating, Payment type, etc.)

### Key Analysis
- Sales performance by **Region, City, Branch, Product Category**
- Trends by **Day, Month, Year** and **Time of day**
- Performance by **Customer type & Payment method**
- Top & bottom performing product lines

### KPIs
- **Total Sales**
- **Total Profit**
- **Quantity Sold**
- **Average Rating**
- **Profit Margin %**

### What I Did
- Cleaned and transformed raw CSV data using **Power Query**
- Built a **star-schema data model** with proper relationships
- Created calculated columns and **DAX measures** for KPIs & time intelligence
- Designed an **interactive dashboard** with slicers, drill-through and tooltips
- Added summary insights for quick decision-making

**Tools & Skills:** Power BI, Power Query, DAX, Data Modelling, Data Cleaning, Data Visualization
